# ЧекиПРО — лендинг

## Запуск

Для интерфейса достаточно любого веб-сервера. Обработчик формы требует PHP 8.1+, расширения PDO MySQL, mbstring и cURL, а также MySQL/MariaDB.

1. Выполните `schema.sql` в нужной базе.
2. Передайте серверу переменные окружения из `.env.example`.
3. Укажите `TELEGRAM_CHAT_ID` — ID пользователя, группы или канала, куда бот должен отправлять заявки.
4. Разместите файлы в корне сайта.

Секреты нельзя добавлять в `script.js`, `index.html` или публичный репозиторий.
